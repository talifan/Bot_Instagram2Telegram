Бот нужен чтобы можно было скачать видео из инсты в телегу как вложение и далее форварднуть или оставить себе. 
Работает с постами и рилзами. С картинками не работает.
Для работы надо сфорировать файл cookie.txt с корректным содержимым:
1. Залогиниться в аккаунт инсты в браузере.
2. В браузере поставить расширение Get cookies.txt LOCALLY
3. Сохранить содежимое куки в файл через Notepad++ с переносом строк CRLF

Поставить все модули в python:
- yt-dlp
- python-telegram-bot

В системе должнен стоять ffmpeg
- sudo apt install ffmpeg

В текст скрипта вписать ключ для бота взятый от Bot Father
